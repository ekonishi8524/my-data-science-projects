# ニューヨーク市内のタクシーデータから読み取るコロナの社会影響
このポートフォリオは、SQL (BigQuery）とPython（Pandas, Seaborn, Matplotlib）を用いたデータ解析コードのノートブック2点です.
新型コロナウィルス感染症（以下、**コロナ**）が及ぼした社会影響を、2020年のニューヨーク市内のタクシーデータを解析することで、読み取ることが目的です.

## 各ノートブックの詳細

### 1. コロナがニューヨーク市に及ぼした社会影響
<ul>
 <li>  <strong>課題・目的</strong>: 2020年のコロナ前後におけるニューヨーク市内のタクシーデータを解析し、コロナが及ぼした社会影響を読み取ること.
 </li>
  <li> <strong>わかったこと</strong>:
<ul>   
  <li>  まず、3月～4月付近（世界的なロックダウンの時期）に、乗車数が劇的に大暴落していることを確認しました.
    <br>
<img src="/corona_and_nyc_taxis/images/corona_1.png" alt="コロナ_1" width="450">
    </li>
  <li>  乗車数が減った（=利用者が減った）だけではなく、1回あたりの平均チップ額も減っていることがわかりました.
    </li>
  <li>  曜日ごとに調べると、コロナ前は平日のチップ率が高く土日は低かったですが、コロナ後は曜日に関わらず一定になりました. このことから、<strong>街からビジネスマンと観光客が消えた</strong>という社会現象が可視化できます.
    <br>
<img src="/corona_and_nyc_taxis/images/corona_2.png" alt="コロナ_2" width="450">
  </li>
   <li>   更に、運転速度が劇的に上がったことがわかり、街から車が消え、ニューヨーク名物の<strong>大渋滞が完全に解消された</strong>ことが読み取れます. 平均移動距離が伸びた (遠出した) こともわかり、このことからは短距離移動でのタクシーの利用を避け、感染リスクを避けようとする乗客の心理が読み取れます.
   </li>
</ul>
  </ul>

### 2. より踏み込んだデータ解析
<ul>
  <li> <strong>課題・目的</strong>: 曜日と時間帯別にまで踏み込んだデータ解析を行いました.
  </li>
  <li> <strong>わかったこと</strong>:
<ul>
    <li>   チップ率が土曜の午前3時と、水曜の午前9時とが突出して高くなり、前者に関して、「土曜午前3時は、乗車件数自体が少ないのではないか?」という仮説を経て、検証しました.
    </li>
    <li>   検証の結果、「10月のavg_tip_rateだけ桁が違う（乗車件数が多くなく、平均値が圧倒的に高い）」という異常値（アウトライアー）を確認しました.
    </li>
    <li>   異常値の原因となるデータを取り除いたところ、乗車件数（Total trips）が5月に底をついたまま年末まで全く回復していない（底をついたまま）のに対し、チップ率（Avg_tip_rate）は5月を底に一度上昇し、元の3分の2程度まで持ち直している、という非対称な動きがわかりました. 前者はリモートワークの完全定着及び観光客の消滅を、後者は乗客の質（属性）が（出勤を免れないエッセンシャルワーカーと、感染リスクを避けるためにあえて地下鉄ではなくタクシーを選ぶ富裕層へ）変化し、激減した乗客を乗せて走る運転手への支援の心理を反映していると考えられます.
      <br>
<img src="/corona_and_nyc_taxis/images/corona_3.png" alt="コロナ_3" width="450">
    </li>
    <li>   このデータベースからはタクシー車両の特定が困難なため、代わりにJFK空港（ロケーションID:132）のタクシー乗り場に焦点を当て、「JFK空港で、前のタクシーが客を乗せて出発してから、次のタクシーが客を乗せるまでの間隔」を月別に出しました. その結果、3月～4月は未曽有のパニックと国際空港の「機能停止」による乗客の消滅、4月～6月はタクシーの数が限界まで縮小した「新常態 (ニューノーマル)」への移行、7月以降は必要最低限の乗客だけによる底を打ったままの横ばい、を読み取ることが出来ました. 以下に、そのグラフを示します.
      <br>
<img src="/corona_and_nyc_taxis/images/corona_4.png" alt="コロナ_4" width="450">
    </li>
</ul>
</ul>
